# Goroutine
非抢占式 由协程自己在切换点交出cpu控制权

Goroutine 可能的切换点

- I/O, select
- channel
- 等待锁
- 函数调用(有时)
- runtime.Gosched()

# 编码相关

## Code

```Golang
func TestRune(t *testing.T) {
	s := "12一二"
	t.Log(len(s))
	t.Log("------")
	for i, b := range []byte(s) { // utf8
		t.Logf("%d %x", i, b)
	}
	t.Log("------")
	for i, b := range s { // utf8 --> unicode
		t.Logf("%d %x", i, b)
	}
	t.Log("------")
	for i, b := range []rune(s) { // utf8 --> unicode --> []rune
		t.Logf("%d %x", i, b)
	}
}
```
